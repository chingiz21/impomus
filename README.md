# impomus

## Description

Import your playlists from Spotify to Youtube

## Used technologies:

- NodeJS
- ExpressJS
- Axios
- React
- React-router
- Redux
- Spotify & Google API

## Folders

- <code>/api</code> - server side
- <code>/browser-extension</code> - extension for Chrome
- <code>/client</code> - client side
